# How to contribute:

- Step 1: Fork this repository.
- Step 2: Clone the repository in your local machine.
- Step 3: Create a new branch.
- Step 4: Create a new `.json` file inside the `./themes/` directory. 
- Step 5: Save the file and commit your changes with the proper message.

```
- Added theme "<Name of your theme>". Made by: <Username>
```

- Step 6: Push your commit to GitHub.
- Step 7: Submit a Pull Request.
- Step 8: All Done.
